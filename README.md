# CloudCraft Organizer

## Overview
CloudCraft Organizer is a modern, cloud-based file management system designed for seamless document storage, categorization, and retrieval. Built on a Next.js framework, it leverages Firebase and Google Cloud Platform (GCP) to provide real-time data synchronization and secure, role-based access for users.

##  Key Features
* **Smart File Management:** Upload, organize, and view files with built-in preview capabilities and an intuitive category system.
* **Real-Time Synchronization:** Powered by a Firebase backend, ensuring that all file uploads, deletions, and organizational changes are instantly synced across all active sessions.
* **Secure Authentication:** Integrated with NextAuth to handle secure user logins and enforce strict role-based access controls (RBAC), ensuring users can only access authorized directories.

##  Tech Stack
* **Frontend:** Next.js, React
* **Backend/Database:** Firebase (Firestore / Storage), Google Cloud Platform (GCP)
* **Authentication:** NextAuth.js
* **Deployment:** Vercel / GCP (Adjust based on your actual deployment)
